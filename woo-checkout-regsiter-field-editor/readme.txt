```

1. In WordPress Admin ‚Üí Plugins ‚Üí Add New, search for ‚ÄúCheckout Field Editor for WooCommerce ‚Äì Checkout Manager‚Äù.

```

== Frequently Asked Questions ==

= Is it compatible with WooCommerce Checkout Blocks? =
Yes. Free supports **Text, Select, Checkbox** in Block Checkout; Classic gets **20+ field types**. [PRO](https://jcodex.com/plugins/woocommerce-custom-checkout-field-editor/) extends Block capabilities further.

= My fields aren‚Äôt showing‚Äîwhy? =
Confirm how your checkout page is built:
- **Classic shortcode** (`[woocommerce_checkout]`): manage fields in **Classic Checkout Fields**.
- **Block Checkout**: manage in **Blocks Checkout Fields** and ensure the **Checkout block** is used on the page.

= Can I reset to default WooCommerce checkout fields? =
Yes‚Äîuse **Reset to Default** to restore the original layout in one click.

= Can I add prices/fees to fields? =
Free: no.  
PRO: yes‚Äîfixed, custom (user-entered), **% of cart**, **% of subtotal** (with/without tax), dynamic per-unit.

= Can I show custom fields on the My Account page? =
Free: checkout only.  
PRO: **Checkout + My Account** (and values persist for future orders).

= How do I add a text field to checkout? =
**WooCommerce ‚Üí Checkout Form Editor ‚Üí Add New Field ‚Üí Text ‚Üí Save.**  
Adjust label, placeholder, required, and position.

= How do I make a Select have no default? =
Add a **blank first option** (empty text & value) and optionally set a placeholder like ‚ÄúSelect an option‚Äù.

= Will this work with any theme? =
Yes, with most well-coded themes. If a theme hard-overrides checkout templates, keep templates updated and avoid conflicting customizations.

= How do I translate field labels and messages? =
Use **WPML, Polylang, or Loco Translate**. All strings are translation-ready.

= Does it support validation? =
Yes‚Äî**Email, Phone, Number, Postcode/State**. PRO adds custom validation options and broader controls.

= Any performance tips? =
Exclude checkout from **JS/CSS minify/deferral** and from **full-page caching**. Don‚Äôt lazy-load critical checkout scripts/styles.

= Can I export/import field setups? =
Free: not available.  
PRO: **Yes**, move configurations between sites easily.

= Is multisite supported? =
Yes‚Äîactivate network-wide or per site as needed.

= Can I add fields conditionally (only for certain products, roles, or payment methods)? =
Free: manual show/hide only.  
PRO: **Conditional Logic** by **role, country, cart totals, shipping/payment method, product/category, other field values**, etc.

= Can I override address formats and include my custom fields in addresses and emails? =
Free: default formats.  
PRO: **Yes**, override address formats in **My Account, Thank You, and transactional emails** (including your custom meta).

= Where are the settings? =
Go to **WooCommerce ‚Üí Checkout Form Editor** (left admin menu).
